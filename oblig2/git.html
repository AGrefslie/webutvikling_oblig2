Git & GitHub

Git er et verktøy for fildeling og versjonskontroll. Git er supert for gruppeprosjekter hvor alle kan laste ned siste versjon av prosjektet, og lett jobbe sammen fra forskjellige steder i verden. Med Git kan man gå tilbake i historikken til prosjektet, og se hva som har blitt endret, og deretter laste ned tidligere versjoner. Dette er praktisk hvis noe har gått galt og du vil gå tilbake og endre.
