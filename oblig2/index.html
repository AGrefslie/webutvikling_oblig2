MathML

MathML er designet for å utrykke mattematiske og vitenskaplige utrykk på nett. MathML er nå en integrert del av HTML5. Det har tidligere vært vanskelig å fremstille mattestykker og formler. Måten det ble gjort før, var å skrive inn mattestykke i en eduqation editor i for eksempel MS Word eller lignende, og så ta et skjermbilde.

SVG

SVG står for Scalable Vector Graphics. SVG er vector-basert grafikk i XML format, og har eksistert siden 1999. Fordelen med vektorgrafikk er at den genererer piksler etter hvor mye man skalerer figuren. Dette vil si at, uansett hvor mye du skalerer bilde vil ikke bilde miste kvalitet.

Git & GitHub

Git er et verktøy for fildeling og versjonskontroll. Git er supert for gruppeprosjekter hvor alle kan laste ned siste versjon av prosjektet, og lett jobbe sammen fra forskjellige steder i verden. Med Git kan man gå tilbake i historikken til prosjektet, og se hva som har blitt endret, og deretter laste ned tidligere versjoner.
